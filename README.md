# AOC2020
Advent of Code Solutions for 2020 and Beyond!

## Data
Create top level directory structure like this to save your data sets. **DO NOT COMMIT YOUR DATASETS**
```
    AOC2020/
    └── data/
        ├── 2015/
        │   ├── day01.dat
        │   ├── day02.dat
        │   ├── day03.dat
        │   └── dayXX.dat
        │
        ├── 2016/
        ├── 2017/
        └── {year}/
            └── dayXX.dat
```

## Code

